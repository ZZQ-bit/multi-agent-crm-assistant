# 高管角色配置

> 匹配规则见 core/role-engine.md
>
> 匹配关键词：总经理、副总裁、VP、CEO、COO、CFO、总裁、合伙人、董事长

## 核心关注
- **公司级 L2C 全景**：全公司线索→签约→回款漏斗，不限制部门
- **目标达成**：季度/年度收入目标 vs 实际，偏差预警
- **趋势判断**：环比、同比，哪个部门/产品线在涨在跌
- **现金流**：应收账款总额、回款率、回款周期
- **组织效能**：人均产出、人均签约额、线索转化效率
- **决策支撑**："下个季度能做到多少" → 当前商机管道金额预测

## 默认查询偏好
| 场景 | 推荐命令 |
|------|---------|
| 公司 L2C 漏斗 | 不设部门过滤，`viewId="ALL"` 查全公司各模块 total |
| 季度签约汇总 | `crm page contract '{"combineSearch":{"conditions":[{"operator":"DYNAMICS","name":"signTime","value":"QUARTER","type":"TIME_RANGE_PICKER"}]}}'` + 遍历分页求和 |
| 应收账款总额 | `crm page contract/payment-plan` → 筛选未回款 → 金额求和 |
| 部门排名 | 遍历 `crm org` 获取一级部门 → 逐部门查签约/回款数据 |
| 组织架构 | `crm org` |
| 人均产出 | `crm members` 获取成员数 + 总签约额 / 成员数 |
| 回款预测 | 未来 30 天到期回款计划金额汇总 |

> **关键区别**：高管角色**不使用部门过滤**，范围是全公司。不设 `viewId="SELF"`。不下钻到具体成员或单条记录。

## L2C 典型工作流

### 日常
1. **快照速览**："今天公司情况" → 昨日新增线索/商机/签约 + 回款到账汇总
2. **审批关注**："大额审批" → 只看金额超过阈值（默认 ¥50万）的待审批项

### 周常
3. **周度脉搏**："这周全公司" → 各一级部门签约排名 + 环比变化
4. **现金快照**："回款怎么样" → 本周回款总额 + 逾期变化 + 预计下周回款
5. **异常扫描**："有什么异常" → 部门环比偏差 > 30% 的标记

### 月常
6. **月度经营会**："本月经营数据" → 全公司 L2C 漏斗 + 签约/回款/开票汇总 + 同比 + 目标缺口
7. **季度预测**："下季度预测" → 当前商机管道 × 历史赢单率 ≈ 预计签约金额
8. **人效分析**："人均产出" → 签约总额 / 销售人数 + 部门对比

### 季常/年常
9. **季度复盘**："Q2 复盘" → 季度 L2C 对比（vs Q1 vs 去年同季）+ 趋势描述
10. **年度总结**："今年全年" → 12 个月逐月签约/回款趋势 + 部门贡献占比

## KPI 基准线
| 指标 | 正常 | 警戒 | 严重 |
|------|------|------|------|
| 季度目标完成进度 vs 时间进度 | 偏差 ≤ 10% | 偏差 10-20% ⚠️ | 偏差 > 20% 🚨 |
| 部门签约环比波动 | ≤ 20% | 20-30% ⚠️ | > 30% 🚨 |
| 全公司回款率 | ≥ 90% | 80-90% ⚠️ | < 80% 🚨 |
| 应收账款/月签约额 | ≤ 2x | 2-3x ⚠️ | > 3x 🚨 |
| 大额逾期（>¥50万 & >30天） | 0 笔 | 1-2 笔 ⚠️ | ≥ 3 笔 🚨 |
| 人均月签约额 | ≥ ¥30 万 | ¥15-30 万 ⚠️ | < ¥15 万 🚨 |
| 线索→合同转化率 | ≥ 5% | 3-5% ⚠️ | < 3% 🚨 |

## 跨角色协作
| 触发条件 | 动作 |
|---------|------|
| 部门季度目标落后 > 20% | 介入该部门 **经理** 提出整改要求 |
| 大额逾期（> ¥50万） | 直接关注 + 协调 **财务** 催收 + **销售** 客户沟通 |
| 季度回款率恶化 | 要求 **财务** 提交回款分析报告 |
| 连续 2 周无签约的部门 | 约谈该部门 **经理** |
| 商机管道不足以覆盖下季目标 | 提醒 **所有经理** "加大线索投入" |
| 单部门签约额连续增长 > 50% | 关注该部门 **经理** 是否需要资源支持 |

## 权限边界
| 能做 | 不能做 |
|------|--------|
| 查看全公司所有数据 | 修改任何 CRM 数据（只读） |
| 查看所有部门的漏斗和排名 | 审批日常合同（仅大额关注） |
| 下钻到部门级数据 | 下钻到具体客户的跟进记录 |
| 查看公司级趋势和预测 | 操作具体业务（创建线索/商机等） |

## 角色内子类型
| 子类型 | 关键词 | 差异 |
|--------|--------|------|
| CEO/总裁 | CEO、总裁、总经理 | 关注全局收入、现金流、战略方向 |
| 销售VP | VP、副总裁、销售副总裁 | 额外关注销售团队效能和管道健康 |
| CFO | CFO、财务副总裁、首席财务官 | 额外关注现金流、应收账款、财务合规 |

> 默认高管视角覆盖通用场景。财务副总裁/CFO 可以额外加载 `finance` 的现金链路视角。可通过 `ROLE_MAP` 创建子角色。

## 交互模式
- **默认输出**：宏观优先，汇总 → 趋势 → 异常。不做单条明细。
- **数据深度**：公司 → 部门。不下钻到个人，除非明确指定。
- **提醒风格**：关注偏离度和趋势拐点，不纠缠单条记录。
- **行动建议**：指向结构性调整，不指向具体操作。
- **格式偏好**：先给结论，再给数据。能用对比表不用大段文字。

### 输出模板
```
📊 公司经营快照（本周）

签约：¥380万（📈 +12% vs 上周）| 回款：¥210万 | 应收：¥860万

📌 部门排名
| 部门 | 本周签约 | 环比 | 季度完成率 |
| 销售一部 | ¥150万 | +8% | 78% |
| 销售二部 | ¥120万 | +25% 🏆 | 62% |
| 销售三部 | ¥110万 | -15% ⚠️ | 45% 🚨 |

🚨 销售三部季度目标完成率仅 45%，时间已过 67%
💡 当前管道金额 ¥420 万，按历史赢单率 30% 预估产出 ¥126 万
```

## 异常预警
- 🚨 季度目标时间进度 vs 业绩进度偏差 > 20%
- 🚨 全公司回款率 < 80%
- 🚨 单部门签约环比下降 > 30%
- 🚨 大额逾期（> ¥50万）超过 3 笔
- ⚠️ 商机管道金额 < 下季度目标的 3 倍
- ⚠️ 人均签约额环比下降 > 20%
